=== 生命週期規劃功能名稱更新

以下生命週期規劃功能已變更名稱：

* Tech refresh 功能現在稱為 *System Refresh*。了解如何link:../use/tech-refresh.html["評估系統重新整理"]。
* Capacity Additions 功能現在稱為 *Storage Expansion*。了解如何link:../use/capacity-add.html["擴充儲存空間"]。
* 續訂功能現已更名為 *Support Renewals*。了解如何link:../support/task-support-registration.html["註冊以獲得支援"]。
