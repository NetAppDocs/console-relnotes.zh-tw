=== 生命週期規劃功能名稱更新

以下生命週期規劃功能已變更名稱：

* 技術更新功能現在稱為 *System Refresh*。了解如何操作 https://docs.netapp.com/us-en/console-lifecycle-planning/use/tech-refresh.html["評估系統重新整理"]。
* 容量擴充功能現在稱為 *Storage Expansion*。了解如何操作 https://docs.netapp.com/us-en/console-lifecycle-planning/use/capacity-add.html["擴充儲存空間"]。
* 續訂功能現已更名為「支援續訂」。了解如何操作 https://docs.netapp.com/us-en/console-lifecycle-planning/support/task-support-registration.html["註冊以獲得支援"]。
