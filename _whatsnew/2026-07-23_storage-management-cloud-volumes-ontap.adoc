=== Azure 智利中部區域的支援

現在可以在 Azure 的智利中部 （chilecentral） 區域部署 Cloud Volumes ONTAP 9.12.1 GA 及更高版本。支援以下部署類型：

* 單節點
* 在單一可用度區域中使用本機備援儲存設備 （LRS） 實現 HA
* 使用區域備援儲存設備 （ZRS） 在多個可用度區域中實現 HA


有關所有支援區域的列表，請參閱 https://bluexp.netapp.com/cloud-volumes-global-regions["Azure 下的全球區域地圖"^]。
