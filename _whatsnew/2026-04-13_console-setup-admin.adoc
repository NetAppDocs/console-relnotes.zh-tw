=== Console 代理 4.6.0

4.6.0 版本同時支援標準模式和受限模式。

此版本的 Console 代理程式包含安全性改進和錯誤修正，以及以下新功能。

.支援對所需磁碟空間進行組態檢查
您可以透過設定檢查來驗證 Console 代理程式所需的磁碟空間是否可用。您可以從 Console 或 Agent 維護主控台執行設定檢查。link:https://docs.netapp.com/us-en/console-setup-admin/task-troubleshoot-agent.html["深入瞭解如何在 Console 代理程式上執行組態檢查"^]

.使用 ISO 升級 vCenter 部署的 Console 代理程式
如果您在 vCenter 中使用 OVA 部署了 Console 代理程式，現在可以透過將 ISO 掛載到 VM 並從 VM 維護主控台執行升級來將其升級到新版本。link:https://docs.netapp.com/us-en/console-setup-admin/task-upgrade-agent.html["了解如何升級 vCenter 已部署的主控台代理程式"^]
