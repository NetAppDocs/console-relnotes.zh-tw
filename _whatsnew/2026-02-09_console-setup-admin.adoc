=== Console 代理 4.4.0

4.4.0 版本同時支援標準模式和受限模式。

此版本的 Console 代理程式包含安全性改善和錯誤修正。



=== NetApp Console管理

本次發布包含以下內容：

.支援 NetApp Backup and Recovery 工作負載（Oracle 和 Microsoft SQL Server）的角色型存取控制
您可以將 NetApp Backup and Recovery 工作負載（Oracle 和 Microsoft SQL Server）與專案建立關聯。指派給專案後，獲指派該專案 NetApp Backup and Recovery 角色的使用者可以使用這些工作負載。目前其他工作負載不支援基於角色的存取。

link:https://docs.netapp.com/us-en/data-services-backup-recovery/br-rbac-setup.html["了解更多關於為 Backup and Recovery 工作負載設定基於角色的存取權限的資訊。"^]
