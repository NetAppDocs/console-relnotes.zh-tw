=== Cloud Volumes ONTAP 支援新的 Azure 區域

現在您可以在馬來西亞西部（malaysiawest）區域的 Azure 中，在單一和多個可用區部署 Cloud Volumes ONTAP 9.12.1 GA 及更高版本。這包括對單節點和高可用性（HA）部署的支援。

有關所有地區的清單，請參閱  https://bluexp.netapp.com/cloud-volumes-global-regions["Azure 下的全球區域地圖"^]。
